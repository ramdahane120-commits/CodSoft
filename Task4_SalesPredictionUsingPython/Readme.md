### 📈 Sales Prediction Using Python – Project Overview

The Sales Prediction Using Python project is a Machine Learning regression task that aims to predict product sales based on advertising expenditures across different media channels such as TV, Radio, and Newspaper. Understanding the relationship between advertising investments and sales performance helps businesses make informed marketing decisions and optimize their advertising budgets.

The main objective of this project is to analyze advertising data, identify the impact of different promotional channels on sales, and build a predictive model capable of estimating future sales. The project follows the complete data science workflow, including data exploration, visualization, preprocessing, model training, and performance evaluation.

Various visualizations are used to understand the relationship between advertising expenditure and sales. A Linear Regression model is then trained to learn patterns from historical data and predict sales values. The model's performance is evaluated using regression metrics such as Mean Absolute Error (MAE), Mean Squared Error (MSE), and R² Score.

### 🎯 Objectives

* Analyze advertising and sales data.
* Explore relationships between advertising channels and sales.
* Perform data visualization and exploratory data analysis.
* Build a machine learning regression model.
* Predict product sales based on advertising expenditure.
* Evaluate model performance using standard regression metrics.

### 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

### 🚀 Expected Outcome

A machine learning model capable of accurately predicting sales based on advertising spending across different media channels. This project demonstrates the practical application of regression techniques and highlights how data-driven insights can support business decision-making and marketing strategies.

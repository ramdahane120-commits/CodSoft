# 🚢 Titanic Survival Prediction

## 📌 Overview

This project uses Machine Learning to predict whether a passenger survived the Titanic disaster based on features such as age, gender, passenger class, fare, and family information.

## 🎯 Objective

* Analyze Titanic passenger data
* Perform data cleaning and visualization
* Build a classification model
* Predict passenger survival

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 📊 Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation

## 🤖 Model Used

**Random Forest Classifier**

## 📈 Evaluation Metrics

* Accuracy Score
* Classification Report
* Confusion Matrix
* Feature Importance

## 🔍 Key Insights

* Female passengers had a higher survival rate.
* First-class passengers were more likely to survive.
* Fare and passenger class strongly influenced survival.

## 🚀 Result

The model successfully predicts passenger survival with good accuracy and identifies the most important factors affecting survival.

# 🎬 Movie Rating Prediction with Python

## 📌 Overview

This project uses Machine Learning to predict movie ratings based on features such as genre, director, actors, year of release, duration, and votes. The goal is to identify factors that influence movie ratings and build a model capable of estimating ratings accurately.

## 🎯 Objective

* Analyze movie dataset information
* Perform data cleaning and preprocessing
* Explore trends through visualizations
* Build a regression model
* Predict movie ratings

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 📊 Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Data Preprocessing
6. Model Training
7. Model Evaluation

## 🤖 Model Used

**Random Forest Regressor**

## 📈 Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score
* Feature Importance Analysis

## 🔍 Key Insights

* Directors and actors significantly impact movie ratings.
* Popular movies with higher votes tend to receive better ratings.
* Genre and release year also influence rating trends.

## 🚀 Result

The model successfully predicts movie ratings using historical movie data and identifies the most influential factors affecting audience ratings.
